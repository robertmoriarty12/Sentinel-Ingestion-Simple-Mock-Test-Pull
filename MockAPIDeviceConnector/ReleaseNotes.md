# Mock API Device Connector Release Notes

## Version 1.0.0 - Initial Release (February 2026)

### Features
- Initial release of Mock API Device Connector
- Support for ingesting device data via REST API
- Custom table for device inventory
- API Key authentication support
- KQL transformation for data normalization

### Data Types
- **MockAPIDevices_CL**: Device inventory data including OS, hostname, and IP address

### Requirements
- Microsoft Sentinel workspace
- Mock API endpoint with device data
- API key for authentication
